# relogiocadc

Este relógio foi criado com o intuito de ser utilizado pelos professores e administração para fins de mostrar a hora em qualquer tipo de ocasião.
Pode ser usado em momentos como provas, apresentações e secretaria. A ideia da criação do projeto foi de ter um relógio com a cara do colégio, com a marca adventista para ser seu.

Para a implementação desse SaaS no seu ambiente, basta apenas rodar esse projeto usando a estenção LIVE SERVER na sua própria máquina, ou então, hospedar com o NODE.JS e abrir usando o WEBSOCKET para abertura do LIVESERVER remotamente.
