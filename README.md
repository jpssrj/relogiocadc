# relogiocadc

Um relógio digital desenvolvido especialmente para a Rede Adventista, pensado para ser exibido na TV da sala de aula durante provas, simulados e momentos de avaliação.
O objetivo é fornecer aos alunos uma visualização clara do horário, mantendo a identidade visual da Rede Adventista e oferecendo um ambiente mais organizado e profissional.

1. Funcionalidades:
- Relógio digital em tempo real

- Design inspirado na identidade visual da Rede Adventista

- Interface limpa para exibição em TVs e projetores

- Ideal para professores e funcionários utilizarem durante avaliações

- Código simples, organizado e fácil de modificar

2. Por que esse projeto existe?
  A necessidade surgiu para que escolas da Rede Adventista pudessem mostrar o horário de forma clara durante momentos de prova, evitando distrações e facilitando a organização do tempo pelos alunos.
O relógio possui visual alinhado ao padrão Adventista, reforçando identidade e profissionalismo no ambiente escolar.

3. Tecnologias Utilizadas
,HTML5
,CSS3
,JavaScript

4. Como usar
Baixe ou clone este repositório:
" git clone https://github.com/jpssrj/relogiocadc "
Abra o arquivo index.html no navegador.
Conecte o computador à TV da sala de aula.
Coloque em tela cheia (F11) para a melhor experiência.
